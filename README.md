# Team Task Manager

A full-stack MERN task management application with authentication, task tracking, status updates, and project management.

## Features

- User Authentication with JWT
- Protected Routes
- Create Tasks
- Update Task Status
- Delete Tasks
- Task Priority Labels
- Persistent Login
- Responsive UI
- MongoDB Database Integration

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- Vite

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Mongoose
